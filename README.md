## android.pulent.test

Android Architecture components Demo App asking iTunes API REST for Collections and Tracks.

This project contains:

- Kotlin Language
- CLEAN Architecture
- Jetpack Navigation
- Databinding
- ViewModel
- LiveData
- Realm (Room in a future)
